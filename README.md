
# Топология: от дырок к алгебраической классификации

Данный курс представляет собой введение в алгебраическую топологию для школьников 9–10 классов через изучение свойств геометрических фигур, инвариантных относительно непрерывных деформаций.

## Содержание курса

Курс охватывает классические разделы топологии:

- **Топологические пространства** — индуцированная топология, непрерывные отображения и гомеоморфизмы
- **Факторпространства и склейки** — построение топологических объектов через отношения эквивалентности
- **Эйлерова характеристика** — числовой инвариант для классификации поверхностей
- **Классификация компактных поверхностей** — полная характеризация замкнутых двумерных многообразий
- **Фундаментальная группа** — алгебраический инструмент для различения топологических пространств
- **Симплициальные гомологии** — вычисление топологических инвариантов через цепные комплексы

Курс строится вокруг интуитивного понимания классификации поверхностей (торов, проективных плоскостей, бутылок Клейна) и их «дырок» как реализации гипотезы Пуанкаре для двумерного случая.

## Инструменты разработки материалов

- **Typst** — верстка конспектов и слайдов с поддержкой математической нотации
- **Manim** — визуализация топологических пространств и непрерывных деформаций
